# Googleの日本語フォント

Type specimen site

## Getting started

More detailed instructions to come.

First, clone the project and `cd` into the directory. Then run:

```sh
npm install
npm start

# Now serving at http://localhost:3000
```

To compile down to HTML, CSS, and JavaScript:

```sh
npm run build
```

To deploy to the staging domain:

```sh
npm run stage
```
